Encadrement : Joseph Chazalon (EPITA) et Jean-Philippe Moreux (BNF)
Equipe : Maxime Boy-Arnould, Benjamin Clene, Yanis Farhat, Vincent Thong et Zoé Sellos.

Modèles utilisés: LayoutParser, Detectron 2 & YOLOv8
